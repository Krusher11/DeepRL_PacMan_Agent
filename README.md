# DeepRL_PacMan_Agent

## Main Files

- **DQNPacmanAgentV2 -** All the methods relating to the Deep RL agent.
- **DQNPacmanParametersV2 -** All game parameters
- **DQNtesting -** Testing bed of Deep RL agent.
